# AiDesign 快速入门指南

## 🎯 5分钟快速上手

### 第一步：创建账户 (1分钟)

1. **访问官网**：打开浏览器访问 [https://aiplatform.com](https://aiplatform.com)
2. **点击注册**：在首页点击"免费注册"按钮
3. **填写信息**：
   ```
   邮箱：your-email@example.com
   密码：至少8位，包含字母和数字
   用户名：您的个性化用户名
   ```
4. **验证邮箱**：查收验证邮件并点击验证链接
5. **完成注册**：登录您的账户

### 第二步：熟悉界面 (2分钟)

#### 主界面布局
```
┌─────────────────────────────────────────────────────────┐
│  [Logo]  [导航菜单]        [搜索] [用户头像]     │
├─────────────────────────────────────────────────────────┤
│ [侧边栏]              │     [主内容区域]        │
│ - AI对话               │     │                 │
│ - 软件管理             │     │   欢迎页面         │
│ - 工作流               │     │   功能介绍         │
│ - 项目管理             │     │   快速开始         │
│ - 设置                 │     │                 │
│                       │     │                 │
└─────────────────────────────────────────────────────────┘
```

#### 关键区域说明
- **顶部导航**：主要功能入口和用户信息
- **侧边栏**：详细功能分类
- **主内容区**：当前功能的详细界面
- **状态栏**：显示系统状态和通知

### 第三步：首次AI对话 (1分钟)

1. **进入AI对话**：点击侧边栏"AI对话"
2. **选择AI模型**：
   - **新手推荐**：智谱GLM-4（中文友好）
   - **创意工作**：豆包Pro（多模态支持）
3. **开始对话**：
   ```
   输入：你好，我想了解一下AiDesign的功能
   
   AI回复：您好！AiDesign是一个智能设计平台，主要功能包括：
   • AI智能对话和代码生成
   • 设计软件集成（Photoshop、AutoCAD等）
   • 工作流自动化
   • 项目协作管理
   您想先了解哪个功能呢？
   ```

### 第四步：连接设计软件 (1分钟)

1. **软件检测**：点击"软件管理" → "扫描软件"
2. **查看结果**：系统会自动检测已安装的设计软件
   ```
   ✅ Adobe Photoshop 2024 - 已连接
   ✅ AutoCAD 2024 - 已连接  
   ❌ Blender 4.0 - 未安装
   ```
3. **手动添加**：如果有软件未被检测，点击"添加软件"手动指定路径

### 第五步：创建第一个项目 (可选，1分钟)

1. **新建项目**：点击"项目管理" → "新建项目"
2. **填写信息**：
   ```
   项目名称：我的第一个AI设计项目
   项目类型：设计项目
   描述：使用AI辅助设计工作流
   ```
3. **完成创建**：点击"创建"开始使用

## 🚀 核心功能快速体验

### AI对话体验

#### 实用示例
1. **设计建议**：
   ```
   我需要设计一个现代风格的Logo，有什么建议吗？
   ```

2. **代码生成**：


3. **文档创作**：
   ```
   帮我写一份产品使用说明文档
   ```

### 高级AI功能

#### 图像生成
1. **进入高级AI**：点击导航栏"高级AI"
2. **选择图像生成**：
   ```
   描述：一只可爱的猫咪在彩虹上，卡通风格
   模型：Stable Diffusion
   尺寸：512x512
   ```
3. **生成图像**：点击"生成图像"等待结果

#### 语音合成
1. **文本输入**：
   ```
   文本：欢迎使用AiDesign智能设计平台
   语音：女声
   语速：正常
   ```
2. **生成语音**：点击"生成语音"并下载音频文件

### 工作流快速开始

#### 创建简单工作流
1. **打开工作流编辑器**：点击"工作流" → "新建工作流"
2. **拖拽节点**：
   ```
   输入 → AI处理 → 输出
   ```
3. **连接节点**：按顺序连接各个节点
4. **测试工作流**：点击"测试"查看运行效果

## 💡 新手技巧

### 提高AI对话效果

1. **具体描述**：
   ```
   ❌ 不好：帮我处理图片
   ✅ 很好：帮我把这张人像照片的背景换成蓝天白云
   ```

2. **提供上下文**：
   ```
   我正在设计一个环保主题的网站，需要设计相关的图标和配色方案...
   ```

3. **分步提问**：
   ```
   第一步：帮我分析这个设计需求
   第二步：基于分析结果给出设计方案
   第三步：提供具体的实现步骤
   ```

### 软件集成技巧

1. **权限设置**：确保设计软件有足够的系统权限
2. **插件更新**：定期更新AI插件以获得新功能
3. **文件备份**：使用AI功能前先备份原始文件

### 工作流优化

1. **使用模板**：从模板库开始，节省开发时间
2. **模块化设计**：将复杂工作流拆分为小模块
3. **错误处理**：添加异常处理节点提高稳定性

## 🛠️ 常见问题快速解决

### 登录问题
**问题**：忘记密码  
**解决**：点击"忘记密码" → 输入邮箱 → 查收重置邮件

**问题**：邮箱收不到验证码  
**解决**：检查垃圾邮件文件夹 → 稍后重试 → 联系客服

### AI功能问题
**问题**：AI回答不准确  
**解决**：重新表述问题 → 提供更多细节 → 尝试不同模型

**问题**：生成速度很慢  
**解决**：检查网络连接 → 简化问题描述 → 切换到快速模型

### 软件集成问题
**问题**：检测不到设计软件  
**解决**：
1. 确认软件已正确安装
2. 以管理员身份运行AiDesign
3. 手动添加软件路径
4. 重启软件

**问题**：插件安装失败  
**解决**：
1. 检查软件版本兼容性
2. 关闭设计软件后重试
3. 检查杀毒软件是否阻止
