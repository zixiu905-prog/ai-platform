## 脚本管理

### 脚本类型

#### JavaScript脚本
适用于Web相关的自动化任务：
```javascript
// 示例：批量重命名文件
function batchRename(files, prefix) {
  files.forEach((file, index) => {
    const newName = `${prefix}_${index + 1}.${file.extension}`;
    file.rename(newName);
  });
  return `成功重命名 ${files.length} 个文件`;
}
```

#### Python脚本
适用于数据处理和机器学习任务：
```python
# 示例：图像质量评估
import cv2
import numpy as np

def assess_image_quality(image_path):
    image = cv2.imread(image_path)
    if image is None:
        return "无法读取图像"
    
    # 计算清晰度
    gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    sharpness = cv2.Laplacian(gray, cv2.CV_64F).var()
    
    return f"图像清晰度评分: {sharpness:.2f}"
```

#### AutoCAD脚本（AutoLISP）
适用于CAD自动化：
```lisp
;; 示例：批量创建图层
(defun create-layers (layer-list)
  (foreach layer-name layer-list
    (if (not (tblsearch "LAYER" layer-name))
      (command "_LAYER" "_M" layer-name "_C" "1" "" "")
    )
  )
  (princ)
)
```

## 常见问题

### 账户相关问题

**Q: 忘记密码怎么办？**
A: 在登录页面点击"忘记密码"，输入注册邮箱，系统会发送重置密码的链接。链接24小时内有效。

**Q: 可以修改注册邮箱吗？**
A: 可以。进入"设置" → "账户设置"，点击"修改邮箱"，需要验证当前邮箱和新邮箱。

**Q: 如何注销账户？**
A: 进入"设置" → "账户设置"，点击"注销账户"。请注意注销后数据无法恢复。

### AI对话问题

**Q: AI回答不准确怎么办？**
A: 可以尝试：
1. 重新表述问题，更加具体
2. 提供更多背景信息
3. 切换不同的AI模型
4. 调整创造性参数

**Q: 为什么AI回复很慢？**
A: 可能的原因：
1. 网络连接较慢
2. 当前请求复杂度高
3. 系统负载较高
4. 所选模型响应时间较长

**Q: 支持哪些语言？**
A: 目前主要支持中文和英文，其他语言的回答质量可能有限。

### 软件集成问题

**Q: 为什么无法连接Photoshop？**
A: 请检查：
1. Photoshop是否正常运行
2. 版本是否受支持（CS6及以上）
3. 是否给予了足够权限
4. 防火墙是否阻止连接

**Q: 可以同时连接多个软件吗？**
A: 可以。平台支持同时连接多个设计软件，但会消耗更多系统资源。

**Q: 操作过程中软件崩溃了怎么办？**
A: 平台会自动检测并尝试重新连接。如果持续出现问题，建议重启软件和平台。

### 工作流问题

**Q: 工作流执行失败了怎么办？**
A: 可以：
1. 查看错误日志，定位失败节点
2. 检查输入参数是否正确
3. 确认相关软件连接正常
4. 尝试分步骤执行

**Q: 如何分享工作流给同事？**
A: 在工作流详情页面点击"分享"，生成分享链接或直接邀请团队成员。

**Q: 工作流可以执行多长时间？**
A: 根据订阅计划不同：
- FREE计划：最长5分钟
- BASIC计划：最长30分钟
- PROFESSIONAL+：无时间限制

### 计费问题

**Q: 令牌用完了怎么办？**
A: 可以：
1. 等待下月自动重置
2. 购买额外令牌包
3. 升级到更高级的计划

**Q: 如何查看消费明细？**
