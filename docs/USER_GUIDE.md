## 🚀 快速入门

### 1. 系统要求

**Web版本要求：**
- 现代浏览器（Chrome 90+, Firefox 88+, Safari 14+, Edge 90+）
- 稳定的互联网连接
- 推荐屏幕分辨率：1920x1080 或更高

**桌面应用要求：**
- Windows 10/11, macOS 10.15+, Ubuntu 20.04+
- 内存：最低 4GB，推荐 8GB
- 存储：至少 2GB 可用空间
- 网络连接（首次使用需要）

### 2. 快速开始

#### Web版本
1. 打开浏览器，访问 AiDesign 平台网址
2. 点击"注册"创建账户
3. 验证邮箱并登录
4. 根据引导完成初始设置

#### 桌面应用
1. 下载对应平台的安装包
2. 运行安装程序
3. 启动应用并登录账户
4. 允许必要的权限请求

### 3. 首次使用流程

```
注册/登录 → 完善个人资料 → 选择订阅计划 → 连接设计软件 → 开始使用AI功能
```

## 🤖 AI助手使用

### 对话界面

#### 基础操作
1. **输入问题**：在输入框中输入您的问题
2. **选择模型**：根据需求选择合适的AI模型
3. **发送消息**：点击发送或按 Enter 键
4. **查看回答**：AI会实时显示回答内容

#### 高级功能
- **语音输入**：点击麦克风图标进行语音输入
- **图片上传**：支持上传图片进行图像分析
- **代码高亮**：代码回答自动语法高亮
- **导出对话**：支持导出对话记录

### 模型选择

| 模型名称 | 特点 | 适用场景 |
|-----------|------|----------|
| 智谱GLM-4 | 强大的中文理解 | 中文问答、文档处理 |
| 豆包Pro | 多模态支持 | 图文混合、创意生成 |
| 本地LLM | 离线使用 | 隐私保护、快速响应 |

### 使用技巧

1. **明确问题**：提供清晰、具体的问题描述
2. **提供上下文**：相关背景信息有助于更好回答
3. **分步提问**：复杂问题可分解为多个步骤
4. **追问确认**：对回答不清楚的地方及时追问

## 💻 桌面应用

### 安装和配置

#### Windows
1. 下载 `AiDesign-Setup.exe`
2. 右键选择"以管理员身份运行"
3. 按照安装向导完成安装
4. 首次启动会进行系统权限配置

#### macOS
1. 下载 `AiDesign.dmg`
2. 打开安装包，拖拽到应用程序文件夹
3. 首次运行需要在系统偏好设置中信任
4. 完成权限配置后正常使用

#### Linux
1. 下载 `AiDesign.AppImage`
2. 添加执行权限：`chmod +x AiDesign.AppImage`
3. 运行：`./AiDesign.AppImage`
4. 根据提示完成系统配置

- **快速访问**：右键托盘图标快速访问功能

#### 本地集成

#### 离线功能

## ❓ 常见问题

### 账户相关

**Q: 忘记密码怎么办？**
A: 在登录页面点击"忘记密码"，输入注册邮箱，按照邮件指引重置密码。

**Q: 如何更改订阅计划？**
A: 登录后进入"账户设置" > "订阅管理"，选择适合的计划进行升级或降级。

**Q: 多设备登录有限制吗？**
A: 免费账户限制同时登录2台设备，付费账户可同时登录5台设备。

### AI功能

**Q: AI回答不准确怎么办？**
A: 尝试重新表述问题，提供更多上下文信息，或选择不同的AI模型。

**Q: 为什么AI回复很慢？**
A: 可能原因：网络延迟、服务器负载高、问题复杂度高。可尝试简化问题或更换模型。

**Q: 如何提高生成内容的质量？**
A: 
1. 提供详细准确的需求描述
2. 添加具体的要求和约束
3. 使用分步骤的方式询问复杂问题
4. 对结果进行追问和优化

### 软件集成

**Q: 检测不到我的设计软件？**
A: 检查以下项目：
1. 软件是否正确安装
2. 是否有管理员权限
3. 手动添加软件路径
4. 重启桌面应用

**Q: 插件安装失败？**
A: 可能的解决方法：
1. 检查软件版本兼容性
2. 关闭设计软件后重试
3. 以管理员身份运行
4. 检查防火墙设置

### 工作流

**Q: 工作流执行失败？**
A: 排查步骤：
1. 检查节点连接是否正确
2. 验证输入数据格式
3. 查看具体错误信息
4. 检查API密钥和权限

**Q: 如何优化工作流性能？**
A: 优化建议：
1. 减少不必要的节点
2. 合并相似的处理步骤
3. 使用并行处理节点
4. 优化数据传输格式

## 🔧 故障排查

### 网络问题

**症状**：无法连接服务器、加载缓慢
**解决方案**：
1. 检查网络连接
2. 尝试切换网络环境
3. 重启路由器
4. 联系网络管理员

### 性能问题

**症状**：应用运行缓慢、卡顿
**解决方案**：
1. 关闭不必要的应用程序
2. 清理系统缓存
3. 更新显卡驱动
4. 增加系统内存

### 文件问题

**症状**：文件丢失、损坏、无法打开
**解决方案**：
1. 检查回收站
2. 使用文件恢复工具
3. 从备份中恢复
4. 联系技术支持

### 桌面应用问题

**症状**：桌面应用无法启动、崩溃
**解决方案**：
1. 以管理员身份运行
2. 检查系统兼容性
3. 重新安装应用
4. 查看错误日志
